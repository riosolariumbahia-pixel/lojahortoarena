# 🌱 SuaLojaJardim360

**A plataforma SaaS mais completa para hortos, floriculturas e garden centers do Brasil.**

CRM com IA, gamificação viciante, WhatsApp integrado e muito mais.

## 🚀 Tecnologias

- React + Vite + TypeScript
- Tailwind CSS
- Supabase (Auth + Database + RLS)
- Zustand (estado)
- Framer Motion (animações)
- Recharts (gráficos)
- Lucide Icons

## 📦 Instalação

```bash
npm install
npm run dev
```

## 🔑 Configuração

Copie `.env.example` para `.env` e preencha:

```
VITE_SUPABASE_URL=https://seu-projeto.supabase.co
VITE_SUPABASE_ANON_KEY=sua-anon-key
```

## 🗄️ Banco de Dados

Execute o SQL em `src/docs/QUICK_SCHEMA.sql` no SQL Editor do Supabase.

## 🌐 Deploy (Vercel)

1. Importe este repositório no Vercel
2. Configure as variáveis de ambiente
3. Deploy automático!

## 📱 Funcionalidades

### Painel do Lojista
- Dashboard zerado com dados reais
- CRM com cadastro de clientes
- Estoque e catálogo de produtos
- WhatsApp com templates e automações
- Link de convite para clientes

### App do Cliente (MeuJardim360)
- Gamificação: XP, moedas, níveis, conquistas
- Adicionar e regar plantas (+XP)
- Diário do jardim com timeline
- Chat IA Jardineiro Virtual
- Scanner IA de plantas
- Loja com favoritos e carrinho
